title: Dr. Cui will give a tutorial on ANDES at the 5th GridFM Workshop at Harvard

Dr. Cui will present a hands-on tutorial titled "Hands-On Dynamic Simulation in
Python with ANDES: Training Data Generation for Grid Foundation Models" at the
[5th Workshop on Foundation Models of the Electric Grid](https://gridfm.org) at
Harvard, March 17-19, 2026.

The tutorial introduces [ANDES](https://github.com/CURENT/andes), an open-source
Python library for symbolic power system modeling and numerical analysis. ANDES
provides over 100 industry-grade models, including synchronous machines, renewable
energy converters, and grid-forming inverters, and supports power flow,
time-domain simulation, and state estimation. Participants will learn how to load
standard cases, run dynamic simulations, programmatically vary operating
conditions and fault scenarios, and extract time-series results for ML workflows.
The session uses interactive Jupyter notebooks and assumes no prior experience
with dynamic simulation.
